## 新增统计信息

### 任务级别
- 处理时间
- 输出长度
- 工具调用次数
- 错误数量

### 会话级别
- 已处理消息总数
- 重试次数
- 连接状态

### 系统级别
- 消费者数量监控
- 队列消息积压监控
- 竞争警告

## 输出级别

1. **INFO**: 常规操作信息
2. **WARNING**: 警告信息（如消费者竞争）
3. **ERROR**: 错误信息（带堆栈追踪）
4. **SUCCESS**: 成功确认信息

## 日志分隔符

- `=` (70个): 主要区块分隔
- `-` (70个): 子区块分隔
- `🔻` (35个): 消息接收标记
- `❌` (35个): 错误标记

## 使用方法

```bash
# 运行抓取器
python doctor_scraper.py

# 发送测试消息
python test_send_message.py
```
